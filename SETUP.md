# 简历填写系统 - 飞书集成配置指南

## 功能概述

本系统实现了简历在线填写功能，支持：
- 应聘渠道信息录入
- 个人资料完整录入
- 教育/工作经历动态添加
- 简历数据自动同步到飞书多维表格

## 表单结构

### 1. 应聘渠道
- 应聘渠道（网络渠道/内部推荐/其他渠道）
- 应聘岗位
- 预计到岗时间
- 岗位性质（全职/兼职）
- 当前状态（在职/离职/应届生/其他）
- 目前月薪
- 期望月薪

### 2. 个人资料
- 姓名（中文）、姓名（英文）
- 性别、出生日期
- 毕业院校、最高学历/专业
- 户籍地、手机、电子邮件
- 婚姻状况
- 现居住地址及邮政编码
- 健康和法律状况

### 3. 其他模块
- 教育/培训经历
- 工作经历
- 个人特质
- 家庭信息
- 本公司亲友
- 紧急联系人

## 飞书集成配置步骤

### 1. 创建飞书企业自建应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 登录后进入「开发者后台」
3. 点击「创建企业自建应用」
4. 填写应用信息

### 2. 配置应用权限

在应用的「权限管理」页面，开通以下权限：
- `bitable:record:write` - 向多维表格中新增记录

### 3. 创建飞书多维表格

在飞书中创建多维表格，添加以下字段：

| 字段名 | 字段类型 | 说明 |

| 应聘岗位 | 文本 | 必填 |
| 预计到岗时间 | 文本 | |
| 岗位性质 | 单选 | 全职/兼职 |
| 当前状态 | 文本 | 在职/离职/应届生/其他 |

### 4. 获取配置信息

#### 4.1 获取 App ID 和 App Secret
在飞书开放平台的应用详情页获取

#### 4.2 获取多维表格 Token 和 Table ID
从多维表格 URL 中提取：
```
https://xxx.feishu.cn/base/[BITABLE_TOKEN]?table=[TABLE_ID]
```

### 5. 配置环境变量

在项目根目录创建 `.env.local` 文件：
```bash
FEISHU_APP_ID=your_app_id
FEISHU_APP_SECRET=your_app_secret
FEISHU_BITABLE_TOKEN=your_bitable_token
FEISHU_TABLE_ID=your_table_id
```

## 技术栈

- **前端**：Next.js 16 + React 19 + TypeScript + shadcn/ui
- **表单**：react-hook-form + zod
- **后端**：Next.js API Routes
- **集成**：飞书开放平台 API
