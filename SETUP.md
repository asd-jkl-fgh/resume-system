# 简历填写系统 - 飞书集成配置指南

## 功能概述

本系统实现了简历在线填写功能，支持：
- 基本信息、联系方式、个人信息的录入
- 教育/工作经历、家庭信息的动态添加
- 简历数据自动同步到飞书多维表格

## 飞书集成配置步骤

### 1. 创建飞书企业自建应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 登录后进入「开发者后台」
3. 点击「创建企业自建应用」
4. 填写应用信息：
   - 应用名称：如"招聘系统"
   - 应用描述：简历收集系统
   - 应用图标：自定义上传

### 2. 配置应用权限

在应用的「权限管理」页面，搜索并开通以下权限：

**必需权限：**
- `bitable:record:write` - 向多维表格中新增记录
- `bitable:record:read` - 读取多维表格记录（可选）

### 3. 发布应用

1. 在「版本管理与发布」页面创建版本
2. 提交审核并发布
3. 发布到企业内部供员工使用

### 5. 获取配置信息

#### 5.1 获取 App ID 和 App Secret

在飞书开放平台的应用详情页：
- 「凭证与基础信息」→ 复制 `App ID` 和 `App Secret`

#### 5.2 获取多维表格 Token 和 Table ID

打开创建的多维表格，URL 格式为：
```
https://xxx.feishu.cn/base/[BITABLE_TOKEN]?table=[TABLE_ID]
```

从 URL 中提取：
- `BITABLE_TOKEN`：多维表格的 Token
- `TABLE_ID`：表格 ID

示例：
```
URL: https://abc123.feishu.cn/base/bascnAbCdEfGhIjK?table=tblxYz123
则：
- BITABLE_TOKEN = bascnAbCdEfGhIjK
- TABLE_ID = tblxYz123
```

### 6. 配置环境变量

在项目根目录创建 `.env.local` 文件（不要提交到 Git）：

```bash
# 飞书应用配置
FEISHU_APP_ID=cli_a1b2c3d4e5f6g7h8
FEISHU_APP_SECRET=your_app_secret_here

# 飞书多维表格配置
FEISHU_BITABLE_TOKEN=bascnAbCdEfGhIjK
FEISHU_TABLE_ID=tblxYz123
```

### 7. 测试提交

1. 访问网站填写简历信息
2. 点击提交
3. 检查飞书多维表格是否新增了记录

## 常见问题

### Q: 提交时报错"飞书应用配置缺失"
A: 检查 `.env.local` 文件是否正确配置了 `FEISHU_APP_ID` 和 `FEISHU_APP_SECRET`

### Q: 提交时报错"飞书多维表格配置缺失"
A: 检查 `.env.local` 文件是否正确配置了 `FEISHU_BITABLE_TOKEN` 和 `FEISHU_TABLE_ID`

### Q: 提交时报错"获取飞书访问令牌失败"
A: 检查 App ID 和 App Secret 是否正确，应用是否已发布

### Q: 提交时报错"添加飞书记录失败"
A: 
1. 检查应用是否有 `bitable:record:write` 权限
2. 检查多维表格字段名是否与配置一致
3. 确认多维表格没有被设置为只读

## 安全建议

1. **不要提交敏感信息**：`.env.local` 文件包含敏感凭证，不要提交到 Git
2. **定期轮换密钥**：建议定期更换 App Secret
3. **最小权限原则**：只授予应用必需的权限
4. **访问控制**：考虑在网站端添加访问限制（如 IP 白名单、验证码等）
